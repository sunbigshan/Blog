# Blog
积土而为山，积水而为海。

#### 引子

除夕前一天，我踏上了回家的旅途，回想2018年，工作忙碌而又充实，但偶尔还是会有焦虑感，虽然目前职位是高级前端，但又总感觉自身很弱，知识掌握的杂而不精，更谈不上以后进阶到资深前端了。

过年期间，我陆陆续续的看了一些有关知识归纳总结的文章，便下定决心开始写博客，工作之余把之前学过的知识归纳总结起来，通过写作构建自己的知识架构体系，同时提高自身的表达能力。
